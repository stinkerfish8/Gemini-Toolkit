# Gemini Toolkit

Enhances the Google Gemini interface with productivity tools and UI improvements.

## Features
* **Message Saver:** Adds a "Save" button to every message to download the text as a .txt file instantly.

## Installation

1. Install a userscript manager such as [Violentmonkey](https://violentmonkey.github.io/) or [Tampermonkey](https://www.tampermonkey.net/).
2. [**Install Gemini Toolkit**](https://github.com/stinkerfish8/gemini-toolkit/raw/main/gemini-toolkit.user.js)

## How to use
After installation, navigate to [Google Gemini](https://gemini.google.com/). A "Save" button will appear under each message. Clicking it will prompt a download of the message content in plain text format.
